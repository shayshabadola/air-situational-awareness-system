# AI-Powered Air Situational Awareness & Decision Support System

## Overview

This project aims to develop an AI-assisted decision support platform for analyzing aircraft surveillance data. The system processes publicly available ADS-B aircraft tracking data to identify unusual flight behavior, analyze airspace connectivity using Graph Theory, and provide actionable insights through an interactive dashboard.

## Objectives

- Analyze aircraft trajectory data.
- Detect anomalous flight behavior using Machine Learning.
- Model airspace connectivity using Graph Theory.
- Visualize aircraft movement on an interactive map.
- Generate risk alerts and operational insights.

## Tech Stack

- Python
- Pandas
- Scikit-learn
- NetworkX
- Plotly
- Folium
- Streamlit
